Ubuntu使用技巧1

一年多来,都在Ubuntu上进行开发.Ubuntu可以说linux发行版中比较好用的一派,虽然比不上[苏西]的绚丽,但是已经足够使用.
Ubuntu是Debian的一个分支,安装软件方便简单.

sudo apt-get install + softname

当然我就把我常常用到的给列出来.

Ubuntu快速卸载软件.

sudo apt-get purge

来自于如下:

apt-get --purge remove ... is the the same as apt-get purge ..... 
purge removes global configuration files only.It does note remove virtual machines,so it's safe to use.

Ubuntu完全卸载libreOffice

sudo apt-get purge libreoffice*
sudo apt-get clean

Ubuntu卸载kazam.

sudo apt-get purge kazam

Ubuntu卸载VirtualBox

sudo apt-get purge virtualbox-*

Ubuntu卸载gnome和gnome-classic

sudo apt-get purge gnome-shell-*

更多详见: 

http://askubuntu.com/questions/65200/remove-gnome-shell-completely-after-installing-it

Ubuntu中删除用户:

sudo userdel -r newuser

参数: -r 删除家目录[还是台湾叫法好啊]
