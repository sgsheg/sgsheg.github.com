关于神的编辑器emacs[1]

关于emacs的使用

  1. 关于emacs在ubuntu 12.04安装

推荐添加ppa:

sudo add-apt-repository ppa:cassou/emacs


sudo apt-get update


sudo apt-get install emacs-snapshot-el emacs-snapshot-gtk emacs-snapshot

更多内容请:

how to install emacs24.3 on Ubuntu

  1. emacs启动

打开emacs

$ emacs

不打开图形界面，直接在终端使用

$ emacs -nw

也可以直接使用的命令来打开一个文件

$ emacs filename.txt

去掉颜色

$emacs -nw --color=no
